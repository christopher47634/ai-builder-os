## 中文项目简介

AI Builder OS 是一个作品集级 AI 工作台。它把原本分散在文档、表格、模板库和评测脚本里的资料整合成一个可运行的 Web 应用。

它包含 7 个核心模块：

- Dashboard：查看整体数据和推荐入口
- AI 项目库：筛选 100 个 AI 项目，生成简历/MVP/作品集文案
- 客户接单：根据客户原话分析需求、风险、报价和 SOW
- 工作流模板：搜索 200 个自动化工作流和 Prompt 模板
- AI Eval Bench：展示 600 条 RAG / Agent / 安全 / 幻觉评测用例
- 作品集包装：生成中英文简历 bullet 和面试讲法
- 验收状态：展示数据源、功能边界和升级路线

当前项目默认离线运行，不需要 API Key。真实 LLM 接入已列入 Roadmap。
